## Frequently Used Code

### Comment
```
# this is a comment
```

### Print Text
```
print("this is text")
```

### convert to string
```
str(number)
```

### convert to integer
```
int(number)
```

### convert to float
```
float(number)
```

### get input
```
num1 = input("give me a number: ")
```

### Lists
```
friends = ["James","Jimmy","Billy"]
print(friends[0])
print(friends[1])
print(friends[2])

newFriends = ["Nic","Sue","Joe"]
friends.extend(newFriends)

print(friends[0,7])
```

### functions
```
def say_hi():
    print("Hi people!")

say_hi()
```
```
def say_hi(name):
    print("Hello " + name)

say_hi("Justin")
```
```
def average(num1,num2,num3):
    return (float(num1) + float(num2) + float(num3)) /3

avg = average(0.1,10.1,5.5)
print(avg)
```
